Sample behavioral waveforms for design file "STACK_POINTER_COUNTER.vhd"

The following waveforms show the behavior of lpm_counter megafunction for the chosen set of parameters in design "STACK_POINTER_COUNTER.vhd". The design "STACK_POINTER_COUNTER.vhd" is a 3 bit dynamically controlled up/down counter. The counter has an asynchronous set signal, that sets the counter to 4. The counter has an asynchronous clear input.
